# 🎵 Greatful Sound

Uma landing page moderna para uma loja fictícia de instrumentos musicais, desenvolvida com **HTML5** e **CSS3**. O projeto apresenta uma interface elegante, organizada e responsiva, oferecendo uma experiência agradável para navegação entre diferentes categorias de produtos.

## 🚀 Sobre o projeto

O **Greatful Sound** foi criado com o objetivo de praticar conceitos de desenvolvimento Front-End, como estruturação semântica em HTML, estilização com CSS e organização de componentes visuais.

A página simula uma loja virtual especializada em instrumentos musicais, apresentando diversas categorias e produtos de forma limpa e intuitiva.

## ✨ Funcionalidades

* 🎸 Página inicial com seção Hero
* 🎵 Menu de navegação com âncoras
* 🛍️ Catálogo organizado por categorias
* 🎹 Cards de produtos com imagem, nome e preço
* 🎨 Layout moderno e agradável
* 📱 Estrutura preparada para adaptação responsiva
* ⚡ Interface simples e de fácil navegação

## 🎯 Objetivo

Este projeto foi desenvolvido para aprimorar habilidades em desenvolvimento Front-End, praticando:

* Estruturação semântica com HTML;
* Organização de layouts;
* Estilização moderna com CSS;
* Criação de componentes reutilizáveis;
* Desenvolvimento de interfaces para e-commerce.
